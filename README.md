The Client Application is the **Health Logger interface** of the *Healthcare Eclipse* platform.

It allows users to **securely log daily health and lifestyle data**, including vital parameters, mood, stress levels, and habits. This data is stored safely and analyzed using AI to detect early signs of **silent diseases**.

### What this app does:
- Provides **login and signup** for clients
- Enables **daily health data logging**
- Tracks **mood and stress patterns**
- Displays **personal health trends**
- Sends data for **AI-based risk analysis**
- Notifies users about **potential health risks**

This application works alongside a **Doctor Dashboard** and a **central backend**, forming a complete preventive healthcare system.
